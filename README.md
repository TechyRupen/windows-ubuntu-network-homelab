# Windows and Ubuntu Networking Homelab

## Overview
This project demonstrates the configuration of a small virtual network using VirtualBox. A Windows 10 VM and an Ubuntu Linux VM were configured with static IP addresses to establish internal network connectivity.

## Lab Environment
Virtualization Platform: Oracle VirtualBox

Operating Systems:
- Windows 10
- Ubuntu Linux

Network Type:
Internal Network

IP Configuration:
Windows: 192.168.10.10
Ubuntu: 192.168.10.20

## Objectives
- Configure static IP addresses
- Enable network communication between Windows and Linux
- Troubleshoot network configuration issues
- Validate connectivity using ping and traceroute

## Key Learning
This lab demonstrates core networking concepts such as IP configuration, virtualization networking modes, troubleshooting connectivity issues, and cross-platform communication.
